# angular-web
This is a starter template for web projects using [Angular](https://angular.io/)

## How to use this template

Required:

* nodejs
* [Angular CLI](https://github.com/angular/angular-cli)

```bash
$ cd angular-web/
$ run `npm install`
$ run `ng serve`
```

start server [http://localhost:4200](http://localhost:4200)
